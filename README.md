# Financial Performance Dashboard (Power BI)

## Overview
This project analyzes financial performance using Power BI, focusing on revenue, profit, and profitability across products, regions, and customer segments.

## Key Metrics
- Total Revenue: $119M
- Total Profit: $17M
- Profit Margin: 14.23%
- MoM Revenue Growth: 11.24%

## Key Insights
- Government segment contributes ~44% of total revenue
- Revenue shows strong growth toward late 2014
- Paseo is the top-performing product by revenue
- North America generates the highest profit

## Features
- Interactive dashboard with slicers
- Dynamic KPIs with conditional formatting
- Time-series analysis of revenue and profit

## Tools Used
- Power BI
- DAX
- Data Modeling

- Power BI report (.pbix)
